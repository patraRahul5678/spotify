## 📖 About the Project

This project is a frontend recreation of Spotify's user interface, designed to mimic the look and feel of the popular music streaming platform.

The primary objective was to gain hands-on experience with frontend technologies while learning how to build responsive layouts, implement media player controls, and create interactive user interfaces using JavaScript.

Through this project, I improved my understanding of web development fundamentals and gained practical experience in recreating real-world applications.

---

## ✨ Features

* 🎵 Spotify-inspired user interface
* 📱 Fully responsive design
* ▶️ Play and pause music controls
* ⏭️ Next and previous track navigation
